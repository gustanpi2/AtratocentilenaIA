Diseñado bajo los lineamientos de Colombia 5.0, MinTIC y la Sentencia T-622

El Desafío del Atrato y sus Comunidades

⚠️

Problema Crítico

Crecientes súbitas incontrolables, contaminación por sólidos y minería ilegal en la cuenca. Falta crónica de sistemas de alerta temprana para comunidades ribereñas aisladas.

🌧️

La Realidad Conectiva

El Chocó es una de las zonas más húmedas del planeta. Los sistemas tradicionales fallan porque no resisten la humedad extrema ni la pérdida constante de energía y conectividad a internet.

Arquitectura de Software Robusta y Escalable

Frontend Core

Desarrollado con React 18 para una interfaz reactiva, Vite para empaquetado ultra rápido y TypeScript garantizando tipado fuerte y cero errores en producción.

Styling & Ergonomía Visual

Tailwind CSS para diseño responsive adaptado a cualquier dispositivo, Lucide Icons y un esquema nativo Dark Mode / Glassmorphism para entornos operativos prolongados.

¿Por qué AtratoCentinela AI es la Solución Ganadora?

Datos Híbridos

Integramos el flujo inmediato de nuestras estaciones de hardware con las APIs de datos abiertos e históricos de CodeChocó e IDEAM.

Simulación y Resiliencia Local

Diseñado para fallar de forma segura (Fail-Safe). Ante pérdidas de internet o energía, la estación almacena los datos localmente y activa alertas analógicas.

Código de Colores Inteligente

Naranja Intermitente = Pérdida de comunicación (Estación en modo offline autónomo).

Rojo Crítico = Anomalía ambiental detectada en el territorio.
